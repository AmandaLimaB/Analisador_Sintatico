# Analisador Sintático

Desenvolver um Analisador Sintático Descendente
Preditivo Recursivo para a gramática definida no problema
2.

Observações:
1. A entrada para este analisador é um conjunto de
arquivos texto. Estes arquivos de entrada deverão
estar em uma pasta na raiz do projeto chamada files.
2. A saída do analisador sintático deverá ser através de
um conjunto de arquivos de saída (um para cada
arquivo de entrada), denominados X-saida.txt, onde X
é o nome do arquivo de entrada. Estes arquivos de
saída deverão estar em uma pasta na raiz do projeto
chamada files.
3. Os arquivos de saída deverão apresentar a lista de
erros sintáticos. Se não houver erros, uma mensagem de sucesso deve ser exibida. O formato dos
arquivos de saída será definido pelo tutor e todos
devem segui-lo.
